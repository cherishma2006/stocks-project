# Stocks Project 

This is a full stack MERN project based on stock market tracking and portfolio management.  
The project allows users to monitor stock prices, manage portfolios, and create watchlists through a modern dashboard UI.

---

# Tech Stack 

## Frontend
- React JS
- Tailwind CSS
- React Router

## Backend
- Node JS
- Express JS
- MongoDB
- Mongoose

---

# Features

- User Registration & Login
- Authentication using JWT
- Dashboard with portfolio details
- Live stock search
- Buy Stocks feature
- Portfolio Management
- Watchlist Feature
- Profit/Loss Tracking
- Profile Page
- Full Stack Deployment

# Environment Variables

## Frontend `.env`

```env
VITE_API_URL=https://stocks-project-uiu8.onrender.com
```

## Backend `.env`

```env
MONGO_URL=your_mongodb_url
JWT_SECRET=your_secret_key
```

# Run Frontend

```bash
cd frontend

npm run dev
```

---

# Run Backend

```bash

```
